## Digital banking introduction

## Basic Environment

NodeJS === `v12.18.0`

## Quick Setup

```bash
// checkout and only execute once
$ cd ant-dashboard
$ npm install

```

## Development

#### Run Webapck for ES6 and React

```bash
// webpack
$ npm start
```

## Testing

```bash
$ npm run test
$ npm run test:coverage
```

## GIT Commit Format

```bash
// for Jira
$ git commit -m "issue {JIRA_ID} MR {MR_ID} Created readme.md file"

```
